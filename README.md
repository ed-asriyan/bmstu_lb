# Авторизатор bmstu_lb
Программа сама авторизует тебя в бауманской Wi-Fi сети, постоянно провернят наличие соединения с интернетом и, если он пропадает, автоматически логинит заново.

*Скоро выложу сюда исполняемые файлы под разные платормы, а пока что программу можно вручную собрать из исходников*

## Создание кофигурационного файла
При первом запуске авторизатор создаст конфиг `bmstu_lb.json` в директории рядом с исполняемым файлом:
```json
{
   "username": "",
   "password": ""
}
```

Заполните вашими логином и паролем от бауманского Wi-Fi, затем запустите авторизатор ещё раз.

## Запуск
```bash
./bmstu_lb
```

## Сборка
```bash
go build -o bmstu_lb -i main.go api.go configuration.go
```
